# lab10z1

Данный код является реализацией алгоритма сортировки пузырьком (bubble sort). Давайте разберём, что делает каждая строка:

1. `void bubble_sort(std::vector<int>& arr)` - Это объявление функции bubble_sort, которая принимает ссылку на вектор целых чисел. Передача по ссылке позволяет нам изменять оригинальный вектор, а не создавать его копию.
2. `bool swapped;` - Объявляется переменная swapped типа bool. Она используется для определения, произошла ли в текущей итерации хотя бы одна перестановка элементов.
3. `int n = arr.size();` - Получаем размер вектора и сохраняем его в переменной n.
4. `for (int i = 0; i < n - 1; i++)` - Внешний цикл, который повторяется n-1 раз. Это необходимо, потому что в худшем случае каждый элемент должен "пробурблиться" через все остальные.
5. `swapped = false;` - В начале каждой итерации внешнего цикла мы предполагаем, что перестановок не произошло.
6. `for (int j = 0; j < n - i - 1; j++)` - Внутренний цикл, который идёт по элементам вектора. При каждой итерации внешнего цикла один из самых больших элементов "всплывает" в конец вектора, поэтому нам не нужно сравнивать его снова, отсюда n - i - 1.
7. `if (arr[j] > arr[j + 1])` - Если текущий элемент больше следующего, то нам нужно их поменять местами.
8. `std::swap(arr[j], arr[j + 1]);` - Меняем местами текущий элемент и следующий.
9. `swapped = true;` - Если мы поменяли элементы местами, то устанавливаем переменную swapped в true.
10. `if (!swapped)` - Если после прохода всего внутреннего цикла перестановок не произошло, значит вектор уже отсортирован.
11. `break;` - Если вектор уже отсортирован, то выходим из внешнего цикла.

В результате работы функции исходный вектор будет отсортирован в порядке возрастания.
