# lab09z2

Давайте рассмотрим каждую строку функции `linearSearchWithBarrier`:

1. `int linearSearchWithBarrier(std::vector<int>& data, int key)` - Это объявление функции `linearSearchWithBarrier`, которая принимает вектор целых чисел (по ссылке, чтобы избежать копирования) и целое число `key`, которое нужно найти.
2. `int n = data.size();` - Эта строка определяет переменную `n` и присваивает ей размер вектора `data`.
3. `int last = data[n - 1];` - Эта строка сохраняет последний элемент вектора `data` в переменную `last`.
4. `data[n - 1] = key;` - Эта строка заменяет последний элемент вектора `data` на искомый ключ. Это "устанавливает барьер" для следующего цикла поиска.
5. `int i = 0;` - Эта строка инициализирует счетчик `i` для цикла поиска.
6. Это цикл, который продолжается до тех пор, пока текущий элемент вектора `data` не равен искомому ключу. Когда ключ найден, цикл останавливается.

    ```cpp
        while (data[i] != key) {
            i++;
        }
    ```

7. `data[n - 1] = last;` - Эта строка восстанавливает значение последнего элемента вектора `data`.
8. Это условное выражение проверяет, был ли найден ключ. Если ключ был найден до последнего элемента (`i < n - 1`) или если ключ равен последнему элементу (`data[n - 1] == key`), то функция возвращает индекс ключа.

    ```cpp
        if (i < n - 1 || data[n - 1] == key) {
            return i;
        }
    ```

9. `return -1;` - Если искомый ключ не был найден в векторе, функция возвращает `-1`.
